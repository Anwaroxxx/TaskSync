### <img src="https://raw.githubusercontent.com/lucide-icons/lucide/main/icons/terminal.svg" width="24" height="24" /> Installation & Setup

1.  **Clone & Enter**
    ```bash
    git clone https://github.com/Anwaroxxx/TaskSync.git
    cd TaskSync
    ```

2.  **Dependencies**
    ```bash
    composer install
    npm install && npm run build
    ```

3.  **Environment**
    ```bash
    cp .env.example .env
    php artisan key:generate
    ```

4.  **Database**
    ```bash
    touch database/database.sqlite
    php artisan migrate --seed
    ```

5.  **Launch**
    ```bash
    php artisan serve
    ```
