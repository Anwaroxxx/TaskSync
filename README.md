# ⚡ TaskSync

> **Professional Task Management Simplified.**  
> A high-performance, minimalist platform built for modern teams. Featuring a premium **pure black** aesthetic and streamlined workflows.

---

### 🚀 Features

*   **🌑 Premium Dark Mode** – Custom pure black theme for maximum focus.
*   **👥 Role-Based Access** – Structured hierarchy for Organizers and Participants.
*   **✅ Real-time Tracking** – Direct lifecycle management of tasks.
*   **📂 Task Organization** – Create, assign, and manage with ease.

---

### 🛠️ Tech Stack

*   **Framework**: [Laravel 11](https://laravel.com) 🏗️
*   **Styling**: [Tailwind CSS](https://tailwindcss.com) 🎨
*   **Database**: [SQLite](https://www.sqlite.org) 📦
*   **Icons**: [Heroicons](https://heroicons.com) ✨
*   **Frontend**: [Blade Templates](https://laravel.com/docs/blade) 🗡️

---

### ⚙️ Installation & Setup

Get started in less than 2 minutes:

1.  **Clone & Enter**
    ```bash
    git clone https://github.com/Anwaroxxx/TaskSync.git
    cd TaskSync
    ```

2.  **Dependencies**
    ```bash
    composer install
    npm install && npm run build
    ```

3.  **Environment**
    ```bash
    cp .env.example .env
    php artisan key:generate
    ```

4.  **Database**
    ```bash
    touch database/database.sqlite
    php artisan migrate --seed
    ```

5.  **Launch**
    ```bash
    php artisan serve
    ```
